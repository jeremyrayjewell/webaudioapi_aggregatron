# WebAudio API Aggregatron

A collection of experimental web audio projects, synthesizers, and browser-based music tools.

Made for the creative coding project [Aggregatron Records](https://aggregatron.neocities.org/).

## Projects

- **Synth Aggregatrex 5000** (`/synth_aggregatrex5000`)
  Feature-rich Web Audio synthesizer with touch, keyboard, and MIDI support.
  Live: https://aggregatron.netlify.app/synth_aggregatrex5000/

- **Synth Aggregatrex 6000** (`/synth_aggregatrex6000`)
  Three.js-based 3D synthesizer interface.

- **Melody Maker Ordnung** (`/melodymaker_ordnung`)
  Algorithmic melody generation organized by runtime complexity.
  Live: https://aggregatron.netlify.app/melodymaker_ordnung/

- **Sample Segmenter Piecework** (`/samplesegmenter_piecework`)
  Audio sample segmentation utility.
  Live: https://bright-capybara-8e9e2c.netlify.app/

- **Preimage Collider Hash-to-Music** (`/preimage_collider_hash-to-music`)
  Turns hash values into musical compositions and generated WAV output.
  Live: https://aggregatron.netlify.app/preimage_collider_hash-to-music/

- **AggregaVox Web Vocoder** (`/aggregavox_web_vocoder`)
  Realtime browser vocoder with a retro instrument-panel interface.

## Local Use

Clone the repo, then run the app you want from its own directory.

React-based projects use their usual install/start workflow.
Static projects can be served locally over HTTP.
